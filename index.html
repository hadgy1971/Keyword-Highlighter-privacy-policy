Privacy Policy

Product: Keyword Highlighter

1. Summary

Keyword Highlighter highlights specified words on webpages for easier reading and analysis. The extension operates fully locally in the user's browser and does not collect, transmit, or store personal data on external servers by default.

2. Local Data Usage

The extension stores only local, non-personal settings required for functionality, such as:

  • Highlighting rules (keywords, colors, options)
  • User preferences (enabled/disabled state, UI choices)

3. No Automatic Data Collection

  • Browsing history is never collected
  • Page content or form input is not sent externally
  • Users are not tracked or profiled

4. Affiliate Banners and Advertising

The extension may display promotional banners or affiliate links in the popup. These banners are clearly labeled as advertisements.

Clicking an affiliate link may generate a commission for the developer. No tracking or profiling is done through these links. External sites are governed by their own privacy policies.

5. Optional Features

Any optional features requiring external connections (cloud sync, backup, analytics) will be opt-in and fully disclosed before any data leaves your browser.

6. Security

All settings and keyword lists are stored locally. The extension requests only minimal permissions (e.g., access to the active tab to apply highlighting). No credentials, cookies, or other sensitive data are transmitted.

7. Data Deletion

Users can delete all settings and keyword lists at any time via the extension UI or browser settings. There is no server-side retention by default.

8. Policy Updates

Updates will be published in this document. The "Last updated" date will change with each new version.
